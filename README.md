# GIS
## 目的
沖縄県那覇市のガソリンスタンドの出店要因を分析することがゴールです。Web上にある出店情報（各店舗の住所）を収集する段階からスタートします。

## コンテンツ
4回で構成されています、基本的にQiitaの記事を見ればわかるようになっていますが、不明点などあれば連絡を頂けると幸いです。
